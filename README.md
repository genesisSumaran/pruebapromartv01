# promart-codequeens
¿Esto es Hostigamiento? - Micrositio Educativo e Interactivo

Este micrositio fue creado por Code Queens, el Squad 14 de la hackatón +ChicasTec de UNICEF. Elegimos el nombre de "Code Queens". El objetivo de este proyecto es informar y educar sobre el hostigamiento laboral en Promart. A través de casos interactivos, una guía paso a paso y funcionalidades dinámicas para registrar reportes, buscamos generar conciencia y proporcionar herramientas de apoyo.

¿Qué puedes encontrar aquí?
Casos Interactivos: Los usuarios pueden leer casos de situaciones laborales y evaluar si constituyen hostigamiento. Después de cada caso, el micrositio proporciona una retroalimentación educativa.
Guía de Acción: Si te encuentras en una situación difícil o no sabes cómo proceder, nuestra guía te acompaña paso a paso para que entiendas qué hacer en caso de hostigamiento.
Sistema de Reporte: Además de aprender y evaluar, los colaboradores pueden reportar situaciones de hostigamiento o simplemente obtener orientación sobre cómo enfrentar el problema.
Andrea, la Asistente Virtual: Un avatar flotante llamado Andrea está presente en todas las páginas para brindarte información adicional. Al pasar el cursor sobre ella, puedes hacer clic y ser redirigido a la página de la ley y protocolo.
Tecnologías Utilizadas
HTML: Para la estructura básica de las páginas.
CSS: Para los estilos y diseño visual. El diseño está basado en la guía de estilo de Promart.
JavaScript: Para manejar la interactividad, como los módulos de casos interactivos y formularios.
